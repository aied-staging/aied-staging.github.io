Open Source Resources To Get Started with K12 AI Education Today

Resources

We have collected a set of websites, projects, and applications that allow children to get started learning about artificial intelligence and related skills. with artificial intelligence tools today. These were developed by members of our team, our collaboartors, and others who are intersted in developing experiences that expose children to artificial intelligence.

App Inventor



App inventor is a visual, block-based programming language for building Android apps.

Learn More

Teachable Machine



Teachable machines is a browser-based platform by Google, where you can train classifiers for your own image recognition algorithm.

Learn More

Scratch



Scratch is a block-based programming language where children 8 to 16 can create their own interactive stories, games, and animations....

Learn More

Snap! AI



Ken Kahn developed a set of extensions to the Snap! programming language to enable children (and non-expert programmers) to build AI programs.

Learn More

Google AI Experiments



AI Experiments is a showcase for simple experiments that make it easier for anyone to start exploring machine learning.

Learn More

Google AIY



With AIY maker kits, build intelligent systems that see, speak, and understand.

Learn More

Google What If



The What-If Tool makes it easy to efficiently and intuitively explore up to two models' performance on a dataset.

Learn More

Tensorflow Playground



TensorFlow Playground is an interactive visualization of neural networks.

Learn More

AI4K12 wiki



This interim site is being used to organize the AI for K-12 initiative jointly sponsored by AAAI and CSTA.

Learn More

The MIT Lincoln Laboratory Beaver Works Center (Lincoln Beaver Works) conducts research and educational programs that strengthen and expand collaborative efforts between Lincoln Laboratory and MIT campus.

Iridescent AI



Iridescent AI Family Challenge is a global competition that challenges families to solve a problem in their community with AI. Families first learn about basic AI concepts through a series of hands-on projects, and then apply what they learn to their own project.

Learn More

GPU Teaching Kit



The GPU Teaching Kit is a set of lessons developed by NVIDIA and Computing in Schools that introduces students to the world of Machine Learning and Deep Learning.

Learn More

Calypso for Cozmo



Calypso is a rule-based programming language for learning about intelligent robots. Calypso was created so that children could write programs for the Cozmo robot by Anki.
